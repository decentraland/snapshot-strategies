# Decentraland Wearable Rarity

This strategy allows users to calculate the total amount of Decentraland wearables and multiply it by rarity

## Example

The space config will look like this:

```json
{
    "symbol": "WEARABLE",
    "multipliers": {
        "unique": 1000000,
        "mythic": 100000,
        "legendary": 10000,
        "epic": 1000,
        "rare": 100,
        "uncommon": 10,
        "common": 1
    }
}
```
